# Ecommerce Full-Stack App

A full-stack ecommerce application with user authentication, item listings, and auctions.

## Structure

- `server/` - Node.js/Express API with MongoDB
- `frontend/` - React/Vite client
- `render.yaml` - Render deployment config
- `DEPLOYMENT.md` - Detailed deployment instructions

## Quick Start

### Backend
```bash
cd backend
npm install
npm start
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

## Deployment

See `DEPLOYMENT.md` for Render and Heroku setup.
